Q: Mention three differences between Web1.0, Web2.0, Web3.0

Difference Web 1.0 Web 2.0 Web 3.0
User Interaction Read-Only, the pages were static and users can search for information and read the pages Read-Write, users can read information and interact with the website by sharing and commenting Read-Write-Execute, Personal and portable, applications are using AI technique (natural language) to suggest the most relvant information for each user.
Primary Streaming Media Home Pages Wikipedia, Blogs Live-Streaming
Advertising Banner Advertising Interactive Advertising Behavioural Advertising


Q: Explain the differences by using use case

The user will search "Eiffel Tower" in the website engine.
Difference Web 1.0 Web 2.0 Web 3.0
User Interaction will provide read-only articles about the Eiffel tower. will provide websites with information about the Eiffel tower' which you can also write your response and get a feedback from the site. will provide information about the Eiffel tower; but also, about other nearby attractions in Paris, cheap flights from your hometown to Paris, recommended hotels in the Eiffel tower area and so on.
Primary Streaming Media The pages that will be provide are home pages with the subject of Eiffel tower The pages that will be provide are Eiffel tower's Wikipedia page and blogs about the Eiffel tower that the user can comment and ask questions The pages that will be provide are youtube's videos on the Eiffel tower, live videos from Paris and the Eiffel tower
Advertising The advertisements that will be shown are banners which will be placed in high-visibility areas on websites The advertisements that will be shown are interactive which will invite the users to physically engage with the ads in an immersive way The advertisements that will be shown are behavioural. Those ads are targeted to users by collecting information about their browsing behavior
